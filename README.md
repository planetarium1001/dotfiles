## Keybindings

Basic Keybindings in this page.

### Hyprland

|Keybinding|Flag|Detail|
|:--|:--:|:--:|
|Super q|-|Close Focused Window|
|Super Shift q|-|Kill Focused Window|
|Super Ctrl Shift Alt q|-|Exit Hyprland|
|Super u|-|Maximize Focused Window|
|Super Shift u|-|Fullscreen|
|Super i|-|Split Mode|
|Super o|-|Floating Mode|
|Super [h, j, k, l]|e|Focus to [Left, Down, Up, Right] Window|
|Super Shift [h, j, k, l]|e|Move Focused Window to [Left, Down, Up, Right]|
|Super Ctrl [h, j, k, l]|e|Resize Focused Window Orienting [Left, Down, Up, Right]|
|Super Shift o|-|Center Focused Window(Floating Only)|
|Super [1-10]|-|Switch to Workspace [1-10]|
|Super Shift [1-10]|-|Move Focused Window to Workspace [1-10]|
|Super [Up, Down]|e|Turn Volume [Up, Down]|
|Super [Right, Left]|e|Turn Brightness [Up, Down]|
|Super Enter|-|Execute `kitty`|
|Super Esc|-|Execute `~/.scripts/settings.sh`|
|Super r|-|Execute `wofi`|
|Super w|-|Execute `firefox`|
|Super s|-|Screenshot(Region) using `hyprshot` and Copy to Clipboard|

More details in [README for Hyperland Keybindings](./docs/hyprland/README.md)

### Theme
All appearance are base on **Tokyonight**  
`Waybar`, `Wofi`, `Mako` are referred to [tokyo-night](https://github.com/lokesh-krishna/dotfiles/tree/main/tokyo-night/).  
`Kitty` used [tokyonight_night.conf](https://github.com/folke/tokyonight.nvim/blob/main/extras/kitty/tokyonight_night.conf) directly.  
`Yazi` used [tokyo-night.yazi](https://github.com/BennyOe/tokyo-night.yazi) directly.  
`Fcitx5` used [fcitx5-Tokyonight](https://github.com/ch4xer/fcitx5-Tokyonight) directly.  
`SDDM` used [tokyo-night-sddm](https://github.com/siddrs/tokyo-night-sddm) directly.
