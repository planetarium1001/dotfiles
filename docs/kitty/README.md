# Keybindings

## Window
|Keybinding|Detail|
|:--|:--:|
|Ctrl h|Focus to Left Window|
|Ctrl j|Focus to Down Window|
|Ctrl k|Focus to Up Window|
|Ctrl l|Focus to Right Window|
|Ctrl Shift h|Move to Left|
|Ctrl Shift j|Move to Down|
|Ctrl Shift k|Move to Up|
|Ctrl Shift l|Move to Right|
|Ctrl u|Resize Window|
|Ctrl Enter|Open New Window|
|Ctrl Shift Enter|Open New Window with Current Work Path|
|Super Shift Enter|Open New OS Window with Current Work Path|
|Ctrl Esc|Close Window|

## Tab
|Keybinding|Detail|
|:--|:--:|
|Ctrl t|Open New Tab|
|Ctrl Shift esc|Close Tab|
|Ctrl i|Switch to Next Tab|
|Ctrl o|Switch to Previous Tab|
|Ctrl Shift i|Move Forward|
|Ctrl Shift o|Move Backward|
|Ctrl Shift t|Set Tab Title|
|Ctrl p|Switch to Next Layout|
|Ctrl Shift p|Switch to Used Layout|

## Font
|Keybinding|Detail|
|:--|:--:|
|Ctrl =|Increase Font Size|
|Ctrl -|Decrease Font Size|
|Ctrl Shift =|Resize Font Size|

## Clipboard
|Keybinding|Detail|
|:--|:--:|
|Ctrl Shift c|Copy to Clipboard|
|Ctrl Shift v|Paste from Clipboard|

## Scrolling
|Keybinding|Detail|
|:--|:--:|
|Ctrl Up|Scrolling Line Up|
|Ctrl Down|Scrolling Line Down|
|Ctrl Shift Up|Scrolling Page Up|
|Ctrl Shift Down|Scrolling Page Down|
|Ctrl Alt j|Scrolling Line Down|
|Ctrl Alt k|Scrolling Line Up|
|Ctrl Shift Alt j|Scrolling Page Down|
|Ctrl Shift Alt k|Scrolling Page Up|
|Ctrl Alt Enter|Scrolling to Bottom|
|Ctrl Shift Alt Enter|Scrolling to Top|
